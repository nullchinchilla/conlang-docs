# Derivational Affixes

| Affix | Type   | Function                | Notes / Examples                                             |
| ----- | ------ | ----------------------- | ------------------------------------------------------------ |
| oQ-   | Prefix | superlative; intensifier | Underlying /Q/ surfaces as `x` [ʔ] before vowels and as gemination before consonants: `oxaatle` “very safe”, `oppaada` “very fat”. |
